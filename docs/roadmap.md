# 개발 로드맵

## v0 — 프로토타입

최소한의 기능으로 빠르게 동작하는 버전을 만듭니다.

### 체크리스트

| # | 작업 | 상태 |
|---|------|------|
| 1 | Strapi에 Term/Category/Tag/Download 모델 생성 | 완료 |
| 2 | Strapi에 Article 모델 생성 | 완료 |
| 3 | Draft/Publish + 역할(RBAC) 구성 | 대기 |
| 4 | Next.js 공개 페이지 뼈대 (용어/카테고리/태그/초성/A-Z) | 완료 |
| 5 | Next.js 메인 페이지 (검색바 + 색인) | 완료 |
| 6 | Next.js 어드민 페이지 뼈대 (/admin) | 완료 |
| 7 | Strapi API → Next.js 연동 (서버 컴포넌트) | 대기 |
| 8 | Meilisearch 검색/자동완성 연결 | 대기 |
| 9 | 로그인 모드 1개 (LOCAL) 구현 | 대기 |
| 10 | /admin 유저/레벨 최소 기능 | 대기 |
| 11 | Docker Compose 개발 환경 완성 | 대기 |

### v0 완료 기준
- Strapi에서 용어를 등록하면 Next.js 공개 페이지에서 조회 가능
- 초성/A-Z 인덱스로 용어 탐색 가능
- Meilisearch로 검색/자동완성 동작
- 이메일+비밀번호 로그인 동작
- /admin에서 유저 목록 조회 가능
